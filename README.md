## Liveblocs - figma clone

### A minimalistic Figma clone based on using fabric.js:

- live collaboration with cursor chat
- comments, reactions, and drawing designs

[Site](https://liveblocks-figma-sxidsvit.vercel.app/)

#### Technologies

- Next.js
- TypeScript
- Liveblocks
- Fabric.js
- Shadcn
- Tailwind CSS
